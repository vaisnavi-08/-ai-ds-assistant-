# 🤖 AI-Powered Data Science Assistant

Predicts Employee Burnout Risk using AutoML + Deep Learning

## 📊 Features
- Auto EDA — automatic data analysis
- Data Cleaning Pipeline
- AutoML — compares 6 ML models automatically
- Neural Network (MLP)
- Interactive Plotly Dashboards
- Model Explainability (SHAP)

## 🏆 Results
- Best Model : AdaBoost
- Accuracy   : 97.99%
- AUC Score  : 0.9987

## 🚀 How to Run
pip install pandas numpy matplotlib seaborn scikit-learn plotly joblib shap
python main.py
